BADM 310 - Introduction to Management and Organizational Behavior Core Course

Instructor: White, J
Semester: fa25
Core Requirement: Yes

Learning Objectives

  • Understand the function of organizational management, and recognize good management
  • Frame an organizational problem as a management problem
  • Select and apply a relevant, evidence-based framework to analyze the problem
  • Generate good, actionable managerial advice consistent with the selected framework
  • Develop the capacity to continue to learn, evaluate, test, and use management frameworks
