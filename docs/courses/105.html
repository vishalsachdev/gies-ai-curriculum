CS 105 - Intro Computing: Non-Tech Core Course

Instructor: Lewis, Colleen M.
Semester: SP26
Core Requirement: Yes

Learning Objectives

  • Master spreadsheet formulas and functions in MS Excel
  • Learn Python programming fundamentals including functions, decision trees, control structures, and classes
  • Develop algorithmic problem-solving and computational thinking skills
  • Apply programming concepts to solve simple real-world problems
  • Understand hardware, software, and network fundamentals
  • Use databases and structured data for practical applications
